# ControlSec Segurança — Site Institucional

Site institucional da **ControlSec Segurança**, empresa especializada em soluções de segurança eletrônica em Fortaleza - CE.

## 📋 Sobre o Projeto

Site desenvolvido em HTML, CSS e JavaScript puro, com uma API em .NET para recebimento de orçamentos.

## 🗂️ Estrutura

```
SiteControlSec/
├── index.html          # Página principal
├── orcamento.html      # Formulário de orçamento
├── Style.css           # Estilos
├── script.js           # Scripts (menu, animações, formulário)
├── orcamento.json      # Dados de orçamentos salvos
├── logomarcacontrolsec.jpg
└── MinhaApi/           # API .NET para receber orçamentos
    └── Program.cs
```

## 🚀 Como Rodar

### Site (front-end)
Abra o `index.html` diretamente no navegador ou use a extensão **Live Server** do VS Code.

### API de Orçamentos (.NET)
```bash
cd MinhaApi
dotnet run
```
A API sobe em `http://localhost:5193`
